Want to explore the code or run this locally?

### Prerequisites
* Node.js 18+
* npm / yarn / pnpm

### Installation

1.  **Clone the repo**
    ```bash
    git clone [https://github.com/amanvermaa01/aman-s-portfolio.git](https://github.com/amanvermaa01/aman-s-portfolio.git)
    cd aman-s-portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Navigate to `http://localhost:3000`.
